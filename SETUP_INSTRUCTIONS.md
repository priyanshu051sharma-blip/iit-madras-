## 🔧 Troubleshooting

### Issue: npm install fails
**Solution**:
```bash
# Clear npm cache
npm cache clean --force

# Try install again
npm install
```

### Issue: Port 3000 already in use
**Solution**:
```bash
# Use different port
npm run dev -- -p 3001

# Or kill existing process
# Windows: taskkill /F /IM node.exe
# Mac: lsof -i :3000 | grep LISTEN | awk '{print $2}' | xargs kill
```

### Issue: Hot reload not working
**Solution**:
```bash
# Restart dev server
# Ctrl+C to stop
# npm run dev to start
```

### Issue: Styles not loading
**Solution**:
```bash
# Clear Next.js cache
rm -rf .next

# Reinstall and restart
npm install
npm run dev
```
